About

Two tools, built to actually be useful.

DevLens is a small set of developer tools: a JSON formatter and validator, and a JWT decoder. Both were built around a simple idea — the tool you reach for ten times a day should be fast, honest about its limits, and shouldn't ask you to sign up.

Why the JSON validator points to an exact location

Most browsers give you a generic "Unexpected token" error when JSON fails to parse, but the exact wording — and whether it includes a line number at all — differs between Chrome, Firefox, and Safari, and has changed across versions. DevLens runs its own lightweight scanner over your text to find a consistent, exact line and column for the first syntax problem, regardless of which browser you're using.

Why the JWT decoder doesn't verify signatures

Verifying a JWT signature requires the secret or public key used to sign it. A tool that asked you to paste that key into a browser page would be a bad security practice, so DevLens simply doesn't offer it. It decodes and displays the header and payload — which anyone can do without a key, since JWTs are only base64-encoded, not encrypted — and clearly labels the signature as unverified.

Your data

Everything you paste into either tool is processed in your browser using JavaScript. Nothing is uploaded, logged, or stored on a server. See our Privacy Policy for details.
