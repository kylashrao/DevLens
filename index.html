Common questions

Is it safe to paste a real JWT or API response here?

Yes — both tools run entirely in your browser's JavaScript engine. Nothing you paste is transmitted to a server, logged, or stored anywhere outside your own device.

Can the JWT decoder verify a token's signature?

No, and intentionally so. Verifying a signature requires the secret or public key used to sign the token, which this tool never asks for. It only decodes the header and payload so you can inspect the claims — it does not confirm the token is authentic or untampered.

Does the JSON formatter handle very large files?

Yes, formatting and minifying work well even on large JSON documents since they use the browser's native JSON parser. Syntax highlighting on extremely large outputs may take a moment longer since every token is individually styled.
